🎮 1. מה זה BLOOM?

BLOOM הוא משחק מיזוג (Merge/Suika) בעברית למובייל. השחקן מפיל אריחים על לוח 6×4 — אריחים זהים מתמזגים לדרגה גבוהה יותר. המטרה: להגיע לכתר (דרגה 8) ולצבור כמה שיותר נקודות.

כל סשן משחק נמשך 3-10 דקות. המשחק תוכנן ליצור לולאת התמכרות: drop → merge → reward → drop שוב.

🎯 3. מצבי משחק

יומי — אותו seed לכל השחקנים. כולם מתחרים על אותו לוח. מופיע בלוח מובילים יומי.

חופשי — seed רנדומלי. כל משחק נספר ונשלח ללוח מובילים. תמיד ניתן לשחק.

תחרות חברים — שחקן יוצר קוד, מזמין חברים. כולם מתחרים על הניקוד הגבוה ביותר. אפשרי עם/בלי הימורים (💎).

אתגר פרס — האדמין יוצר אתגר עם פרס אמיתי. 3 סוגים: race (ראשון שמגיע), top_n (N מובילים), beat (כולם מעל הרף).

אתגר שבועי — נוצר אוטומטית כל 7 ימים. פרס 500💎 (ניתן לשינוי: weekly_prize).

דו-קרב 1v1 — שחקן מאתגר חבר. שניהם מקבלים אותו seed. הזוכה לוקח הכל.

🎯 4. מערכת Events

כל 15-35 שניות מופיע אלמנט מיוחד על תא ריק בלוח עם טיימר. השחקן צריך להפיל אריח לעמודה לפני שהזמן נגמר.

Eventטיימראפקטמפתח הפעלה
💣 פצצה8sמפוצץ 4-8 אריחים + 2,000/אריחevent_bomb_enabled
⭐ כוכב6sמקפיץ אריח דרגה +1event_star_enabled
🎁 מתנה10s5-50💎 (5% JACKPOT 500💎)event_gift_enabled
🔥 Fever5s×3 ניקוד למשך 10 שניותevent_fever_enabled
❄️ הקפאה4sמנקה שורה (הצלה!)event_freeze_enabled
🎯 מטרה12s×5 על מיזוג דרגה ספציפיתevent_target_enabled

תזמון: events_start_delay = שניות עד event ראשון. events_min_gap / events_max_gap = מרווח בין events.

הסתברויות: כל event יש weight. הסתברות = weight / סכום כל ה-weights. לדוגמה: bomb=25, star=20, gift=25, fever=12, freeze=8, target=10 → סה"כ 100 → bomb = 25%.

לכבות event ספציפי: שנה event_X_enabled ל-false או event_X_weight ל-0.

❄️ Freeze מיוחד: מופיע רק כשיש ≥3 שורות מלאות (event_freeze_min_filled_rows).

🧠 6. פסיכולוגיית השחקן

0-30 שניות: שחקן לומד. מפיל אריחים ראשונים. מיזוגים קטנים. רואה ניקוד עולה.

30s-2 דקות: מגיע לדרגות 4-5. בונוס ראשון +500 → "WOW!". Events מתחילים להופיע. מתח — הטיימר על הEvent.

2-5 דקות: מגיע לדרגות 6-7. +5,000 בונוס, screen shake, milestone 50K! Fever mode ×3. השחקן מרגיש חזק.

5-10 דקות: כתר ראשון! +15,000! Crown merge אם מגיע ל-2 כתרים → 💥 פיצוץ שורה + 50,000 + רעידת אדמה! השחקן צועק.

Game Over: כותרת דינמית (משחק אדיר! / כמעט שיא!). "עוד 3,200 נקודות לנצח את דני" → לוחץ "שחק שוב" מיד. בונוס מחר → סיבה לחזור.

🔄 9. מנגנוני שימור

רצף יומי: שחקן שחוזר כל יום מקבל streak עולה (🔥). בונוס גדל: 25💎 → 50💎 → 200💎 (30 ימים).

אתגר שבועי: תחרות שבועית אוטומטית עם 500💎 פרס. שחקנים מתחרים על מקום ראשון.

לוח מובילים: כל משחק (כולל חופשי) מופיע בלוח. שחקן רואה "עוד X נקודות למקום Y" → מוטיבציה.

XP ורמות: 11 רמות (מתחיל → אלמוותי). XP נצבר ממשחקים, streaks, תחרויות.

הישגים: 15 achievements (10K/50K/100K, שרשרת ×3/×5, 3/7/30 ימים ברצף).

Daily login: מודל reward בכניסה הראשונה ליום.

הזמנת חברים: WhatsApp invite עם לינק. מזמין + מוזמן מקבלים 💎.

⏰ 9. תזמון מומלץ

Events: event ראשון אחרי ~15 שניות. event חדש כל 15-35 שניות. לשנות: events_start_delay, events_min_gap, events_max_gap.

Milestones: 10K (~1 דקה), 25K (~2 דקות), 50K (~3 דקות), 100K (~5 דקות). אלה "WOW moments" שמעודדים להמשיך.

אתגר שבועי: מתחדש אוטומטית כל 7 ימים (בדיקה כל שעה). לשנות: weekly_prize, weekly_name.

Fever mode: ×3 ניקוד למשך 10 שניות (event_fever_duration). מספיק ל-3-5 מיזוגים במהירות.

💰 11. מונטיזציה

Continue (חיים נוספים): כשהמשחק נגמר (ניקוד מעל 5K), השחקן יכול:

  • ▶️ צפה בפרסומת → ממשיך לשחק (2 שורות עליונות נמחקות)
  • 💎 שלם continue_price (200💎) → ממשיך לשחק

Watch Ad: בכל game-over, השחקן יכול לצפות בפרסומת ולקבל ad_watch_reward (30💎). קולדאון: ad_cooldown_seconds (30).

איזון כלכלה: הכנסה חופשית ~295💎/יום (בלי פרסומות). פריט premium: 300💎 (כתר). המשמעות: שחקן צריך לשחק ~יום שלם או לצפות בפרסומת כדי לקנות כתר.

⚙️ 12. מפתח הגדרות מלא

כל ההגדרות נמצאות ב"הגדרות משחק → כל ההגדרות" למטה. שינויים נכנסים לתוקף מיידית.

קבוצות הגדרות:

  • merge_mode — כיוון מיזוג (anchor/classic/smart)
  • crown_merge_* — Crown Merge on/off + בונוס
  • events_* — מערכת Events (6 מפתחות כלליים)
  • event_bomb/star/gift/fever/freeze/target_* — הגדרות לכל event (4-7 מפתחות כל אחד)
  • shake_* — עוצמת רעידות (7 מפתחות)
  • daily/streak/contest_*_reward — תגמולי 💎
  • wager_* — הימורים
  • referral_* — הפניות
  • tile_price_* — מחירי חנות
  • weekly_* — אתגר שבועי
