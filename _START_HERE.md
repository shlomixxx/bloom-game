# שלומי — קרא אותי ראשון

זה ההסבר הקצר. הכל פשוט.

## מה הקבצים שאני נותן לך

```
bloom-merged/
├── _START_HERE.md          ← אתה קורא את זה עכשיו
├── server.js               ← השרת המאוחד החדש
├── schema.sql              ← הסכמה המאוחדת החדשה
└── CLAUDE_INSTRUCTIONS.md  ← הוראות לקלוד ב-Cursor
```

## מה המצב הנוכחי שלך

יש לך פרויקט שעובד מצוין:
- שרת ב-Railway (`server.js`, `db.js`, `package.json`, `schema.sql`)
- משחק ב-`public/index.html` עם מצב יומי ולוח מובילים גלובלי

הכל עובד. **אל נשבור את זה.**

## מה אנחנו מוסיפים

תחרויות חברים פרטיות. סבא יוצר תחרות, מקבל קוד 6 תווים, שולח לוואטסאפ, נכדים מצטרפים. **בנוסף** למצב היומי הקיים, לא במקום.

## הצעדים שלך — 3 בלבד

### צעד 1 — החלפת 2 קבצים בפרויקט

תפתח את התיקיה של הפרויקט שלך (זאת שיש בה את `server.js`, `db.js`, `package.json`).

**תחליף 2 קבצים** בקבצים החדשים שאני נתתי לך:

1. **`server.js`** — תמחק את הקיים, תעתיק את החדש במקומו
2. **`schema.sql`** — תמחק את הקיים, תעתיק את החדש במקומו

**אל תיגע**:
- `db.js` — נשאר זהה
- `package.json` — נשאר זהה
- `public/index.html` — קלוד ב-Cursor יטפל בו בצעד הבא

ה-`server.js` החדש כולל את כל ה-endpoints הישנים שלך **בלי שינוי** (`/api/score`, `/api/leaderboard/:date`, `/api/leaderboard/range/:period`) + 4 endpoints חדשים לתחרויות.

ה-`schema.sql` החדש כולל את טבלת `daily_scores` הישנה שלך **בלי שינוי** + 2 טבלאות חדשות. `db.js` ירוץ את הסכמה בהפעלה הבאה ויצור את הטבלאות החדשות אוטומטית.

### צעד 2 — נתינה לקלוד ב-Cursor

תפתח את `CLAUDE_INSTRUCTIONS.md`, תעתיק את **כל** התוכן שלו, ותשים בקלוד ב-Cursor (Cmd+L → הדבק → Enter).

קלוד ב-Cursor יעדכן את `public/index.html` (יוסיף את הפיצ'ר של תחרות חברים).

תיתן לו לעבוד. אם הוא שואל אותך משהו — תספר לי.

### צעד 3 — פריסה ל-Railway

אחרי שקלוד ב-Cursor סיים, בטרמינל בתיקיית הפרויקט תרץ:

```bash
git add .
git commit -m "Add friends contest feature"
git push
```

Railway יראה את ה-push ויפרוס לבד תוך 1-2 דקות.

## בדיקה — איך אדע שזה עובד

תפתח את האתר. במסך הבית אמורים להופיע **3 כפתורים** במקום 2:

```
[ בוא נתחיל ]
[ תחרות חברים   חדש ]     ← זה החדש
[ אני יודע לשחק, דלג ]
```

תלחץ על "תחרות חברים":
- תיצור תחרות חדשה
- תקבל קוד של 6 תווים
- תוכל לשתף לוואטסאפ או להעתיק קישור
- תוכל לראות את לוח התחרות

תוודא שגם **המצב היומי** עדיין עובד — תלחץ "בוא נתחיל", תשחק, תראה את לוח המובילים הגלובלי.
