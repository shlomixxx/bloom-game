## 1. Architecture (one screen)

```
Browser  →  public/index.html (variant loader: reads /api/flags/game_v2, boots app.js or v2)
              ├── public/app.js   (ONE IIFE: 01-constants opens … 99-close closes)
              └── public/styles.css
                     │  fetch /api/* (same-origin, API_BASE='')
                     ▼
Railway bloom-web (Node 18 / Express, server.js)
   ├── public routes (174)  — requireDeviceAuth (HMAC token) on all money/state POSTs
   ├── adminRouter (100)    — mounted at ADMIN_PATH, Basic-Auth (requireAdmin)
   ├── cron/setInterval jobs (bots, tournaments, guild wars, rivalries, bank interest, cleanup)
   └── db.js (pg Pool)
                     ▼
Postgres-z2RQ  (74 tables; game_config is a KV store for settings AND `_`-prefixed dedup keys)
```

Identity: anonymous `bloom_device_id` (localStorage) + HMAC `bloom_device_token` from `/api/register`. No login.
Daily seed: `mulberry32(hashSeed('bloom-YYYY-MM-DD'))` in **Asia/Jerusalem** (fairness contract — never change to UTC).

### 2.4 Config system
`game_config` is a KV table. `loadConfig()` filters out `_`-prefixed keys (dedup junk) into a 60s in-memory `_configCache`. `getCachedConfigPrefix('bank_'/'pet_'/'gacha_'/…)` returns a prefix slice from cache (zero DB round-trip). Admin `PATCH /config/:key` busts the cache. `_`-prefixed rows are per-device/per-day dedup markers (`_earn:`, `_ad:`, `_chest:`, `_trophy:`, `_mig_*`).

### 3.4 Response-shape convention gotcha
Transformed endpoints emit **camelCase** (dashboard, economy `sinkRatioPct`, issues/stats `openCount`, bot-duels/stats, push/status). The admin UI reads each correctly today; keep this in mind when adding panels.

## 6. Fast "where do I…" index
- Change a reward amount → admin 🎮/💰 config editor → `game_config` key (`PATCH /config/:key`); tips in `admin/index.html` `TIPS_PER_KEY`/`PRESETS_PER_KEY`.
- Add a home tile → new `src/NN-*.js` inner IIFE exposing `window.maybeShowXTile`, mount it in `05a-home-v2.js` deferred block, route it in `46-bottom-nav.js` `TILE_TO_TAB`.
- Add an admin panel → add a `<div class="admin-tab-panel" data-tab="…">` + `api('/route')` (single `/api` prefix!) + server `adminRouter.get('/api/route')`.
- Game-over reward not firing → check WHICH branch (3612 vs 3951) in `src/11-game.js` — they're asymmetric.
- Escape user text → `escapeHtml()` (hoisted global) or per-module `esc`/`escDuelHtml`. Always escape names/messages before innerHTML.
- Verify live → `curl $BASE/api/health` + register a test device then hit state/mutation endpoints (all return `{ok:true}` when healthy).
