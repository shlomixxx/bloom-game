# 📖 BLOOM — מדריך מנהל מלא

## 🎮 1. מה זה BLOOM?

BLOOM הוא משחק מיזוג (Merge/Suika) בעברית למובייל. השחקן מפיל אריחים על לוח 6×4 — אריחים זהים מתמזגים לדרגה גבוהה יותר.

**המטרה:** להגיע לכתר (דרגה 8) ולצבור כמה שיותר נקודות.

כל סשן משחק נמשך 3-10 דקות. המשחק תוכנן ליצור **לולאת התמכרות**: drop → merge → reward → drop שוב.

|---|-----|----------------|------------------|-------------|

**Crown Merge:** שני כתרים 👑+👑 = 💥 פיצוץ שורה שלמה + 50,000 נקודות!
- הגדרות: `crown_merge_enabled` (true/false), `crown_merge_bonus` (50000)

**נוסחת ניקוד:**
```
points = tier × 10 × (1 + (tier-1) × 0.3) × groupSize × chainMultiplier
```
Chain multiplier: ×1.0, ×1.5, ×2.0, ×2.5...

## 🎯 3. מצבי משחק

### יומי
- אותו seed לכל השחקנים — כולם מתחרים על אותו לוח
- מופיע בלוח מובילים יומי
- שחקן מקבל 10💎 על סיום (`daily_reward`)

### חופשי
- Seed רנדומלי, כל משחק נספר ונשלח ללוח מובילים
- תמיד ניתן לשחק, אין הגבלה

### תחרות חברים
- שחקן יוצר קוד 4 ספרות, מזמין חברים דרך WhatsApp
- כולם מתחרים על הניקוד הגבוה ביותר
- אפשרי עם/בלי הימורים (💎)
- תחרות פעילה 7 ימים

### אתגר פרס
- **האדמין** יוצר אתגר עם פרס אמיתי (טקסט חופשי)
- 3 סוגים:
  - `race` — ראשון שמגיע לרף ניקוד זוכה
  - `top_n` — N המובילים זוכים
  - `beat` — כל מי שעובר את הרף זוכה

### אתגר שבועי
- נוצר אוטומטית כל 7 ימים (בדיקה כל שעה)
- פרס: 500💎 (ניתן לשינוי: `weekly_prize`)
- שם: `weekly_name`

### דו-קרב 1v1
- שחקן מאתגר חבר עם הימור 💎
- שניהם מקבלים אותו board seed (הוגן!)
- הזוכה לוקח הכל

## 🎯 4. מערכת Events (אירועים מיוחדים)

כל **15-35 שניות** מופיע אלמנט מיוחד על תא ריק בלוח עם טיימר. השחקן צריך להפיל אריח **לעמודה** של ה-event לפני שהזמן נגמר.

### 6 סוגי Events

| Event | טיימר | אפקט | הסתברות |
|-------|--------|------|---------|
| 💣 פצצה | 8s | מפוצץ 4-8 אריחים + 2,000/אריח | 25% |
| ⭐ כוכב זהב | 6s | מקפיץ אריח דרגה +1 | 20% |
| 🎁 מתנה | 10s | 5-50💎 רנדומלי (5% JACKPOT 500💎!) | 25% |
| 🔥 Fever | 5s | ×3 ניקוד למשך 10 שניות | 12% |
| ❄️ הקפאה | 4s | מנקה שורה עליונה (הצלה!) | 8% |
| 🎯 מטרה | 12s | ×5 על מיזוג דרגה מסומנת | 10% |

### הגדרות תזמון

| מפתח | ברירת מחדל | תיאור |
|------|------------|-------|
| `events_enabled` | `true` | מתג ראשי — הפעלה/כיבוי |
| `events_start_delay` | `15` | שניות עד event ראשון |
| `events_min_gap` | `15` | מינימום שניות בין events |
| `events_max_gap` | `35` | מקסימום שניות בין events |
| `events_min_empty_cells` | `4` | מינימום תאים ריקים |

### הגדרות לכל Event

כל event יש:
- `event_X_enabled` — true/false
- `event_X_weight` — הסתברות (גבוה = תכוף יותר)
- `event_X_timer` — שניות לפני שנעלם

הגדרות ספציפיות:
- 💣: `event_bomb_radius` (1=4 שכנים), `event_bomb_points_per_tile` (2000)
- ⭐: `event_star_upgrade` (1), `event_star_points` (500)
- 🎁: `event_gift_credits_min/max` (5-50), `event_gift_jackpot_chance` (5%), `event_gift_jackpot_amount` (500)
- 🔥: `event_fever_duration` (10s), `event_fever_multiplier` (3)
- ❄️: `event_freeze_clear_rows` (1), `event_freeze_points` (1000), `event_freeze_min_filled_rows` (3)
- 🎯: `event_target_multiplier` (5)

### חישוב הסתברויות

```
bomb_weight=25 + star_weight=20 + gift_weight=25 + fever_weight=12 + freeze_weight=8 + target_weight=10 = 100

bomb  = 25/100 = 25%
star  = 20/100 = 20%
fever = 12/100 = 12%
...

לשנות: הגדל weight = יותר תכוף. weight=0 = לא יופיע לעולם.
```

## 🧠 6. פסיכולוגיית השחקן — מה קורה בכל שלב

### 0-30 שניות: למידה
- שחקן מפיל אריחים ראשונים
- מיזוגים קטנים (אבן+אבן=עלה)
- רואה ניקוד עולה לאט
- **הרגשה:** "מעניין, פשוט"

### 30 שניות - 2 דקות: התרגשות ראשונה
- מגיע לדרגות 4-5
- **בונוס ראשון +500 → "WOW!" + רעידת מסך**
- Events מתחילים להופיע → **מתח** (הטיימר!)
- **הרגשה:** "וואו, זה נותן בונוסים!"

### 2-5 דקות: שיא הריגוש
- מגיע לדרגות 6-7
- **+5,000 בונוס, screen shake, milestone 50K!**
- Fever mode ×3 → ניקוד מטורף
- Gift events → 💎 מצטברים
- **הרגשה:** "אני חייב להמשיך, אני כמעט שם!"

### 5-10 דקות: אקסטזה
- כתר ראשון! **+15,000!**
- Crown merge → **💥 פיצוץ + 50,000 + רעידת אדמה!**
- Milestone 100K → "💎 100K!" + 50💎
- **הרגשה:** "אין מצב שאני עוצר עכשיו!"

### Game Over: הרגע הקריטי
- כותרת דינמית:
  - 100K+ → "🔥 מטורף!"
  - שיא חדש → "🎉 שיא אישי חדש!"
  - 90% מהשיא → "😱 כמעט שיא!"
- **"עוד 3,200 נקודות לנצח את דני"** → מוטיבציה
- **"חזור מחר ל-50💎 בונוס"** → סיבה לחזור
- **כפתור "שחק שוב" גדול** → restart מיידי
- **הרגשה:** "עוד משחק אחד..." (×5)

## 📳 7. רעידות מסך (Screen Shake)

כל רעידה ניתנת לשליטה. ערך 0 = כבוי.

| אירוע | עוצמת ברירת מחדל | מפתח config |
|-------|-----------------|-------------|
| מיזוג דרגה 5-6 | 3-4 | `shake_tier_up` |
| מיזוג דרגה 7-8 | 5 | `shake_tier_up` |
| Crown Merge | 8 | `shake_crown_merge` |
| Score milestone | 2 | `shake_milestone` |
| Triple/Quad merge | 3-6 | `shake_multi_merge` |
| שיא חדש | 4 | `shake_new_best` |
| 💣 פיצוץ | 6 | `event_bomb_shake` |
| ❄️ הקפאה | 4 | `event_freeze_shake` |

**המלצות:**
- 2-4 = עדין, מתאים לאירועים תכופים
- 6-8 = דרמטי, מתאים לאירועים נדירים
- 10+ = מוגזם, רק לאפקטים חד-פעמיים

## 🔄 8. מנגנוני שימור (Retention)

### רצף יומי (Streak)
- שחקן שחוזר כל יום → מונה streak עולה (🔥)
- בונוס גדל: יום 1 = 25💎 → יום 3 = +15💎 → יום 7 = +50💎 → יום 30 = +200💎
- ביום game-over: "🔥 3 ימים ברצף! חזור מחר ל-50💎"

### אתגר שבועי
- תחרות אוטומטית עם 500💎 פרס
- באנר זהוב על מסך הבית

### לוח מובילים
- כל משחק (כולל חופשי) נשלח ללוח יומי
- שחקן רואה "עוד X נקודות למקום Y" → מוטיבציה

### XP ורמות (11 רמות)
1. מתחיל (0 XP)
2. מתחיל+ (50)
3. טירון (150)
4. חובבן (500)
5. שחקן (1,200)
6. שחקן+ (2,000)
7. מקצוען (5,000)
8. מומחה (10,000)
9. אלוף (25,000)
10. אגדה (50,000)
11. אלמוותי (150,000)

### הישגים (15)
- 3 דרגות: הגע לאש, כוכב, כתר
- 3 שרשראות: ×2, ×3, ×5
- 3 ניקוד: 10K, 50K, 100K
- 3 רצף: 3, 7, 30 ימים
- 3 כלליים: 1, 10, 50 משחקים

### הזמנת חברים
- WhatsApp invite עם לינק
- מזמין מקבל: `referral_reward` (50💎)
- מוזמן מקבל: `referred_bonus` (50💎)
